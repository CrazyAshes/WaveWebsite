# WAVE 源素材（不参与网站部署）

此目录存放原始文档、备用图片、3D 模型等，**不会**被打包进 `out/` 或上传到 Cloudflare。

## 目录说明

| 目录 | 内容 |
|------|------|
| `documents/` | 原始 PDF、PPT（HCBC、YES Program 等） |
| `images/source/` | 根目录散落的源 PNG |
| `images/instagram/` | 社交媒体导出图 |
| `images/duplicates/` | 与 `public/wave-assets/ai/` 重复的备份 |
| `icons/` | 黑白/蓝色 icon 源文件 |
| `models/` | STL 3D 模型 |
| `archive/wave-assets-legacy/` | 旧版网站图片（JPG/早期 PNG） |
| `archive/pptx-extracts/` | PPT 解压出的资源文件夹 |
| `archive/gallery.html` | 旧版图片预览页 |

## 网站实际使用的资源

见 `public/` 与 `src/lib/assets.ts`。
